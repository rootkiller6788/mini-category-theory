# Coverage

## Implemented

- Monad structure (T, η, μ, with laws)
- MonadMorphism with unit and multiplication compatibility
- Kleisli category Kl(M)
- Monad laws as propositions (left unit, right unit, associativity)
- Theorem: monad laws hold from structure
- Eilenberg-Moore algebra EMAlgebra
- Eilenberg-Moore category EMCat
- Theorem: Every adjunction induces a monad (fromAdjunction)
- Maybe type (conceptual monad)
- List monad (conceptual)

## Stubs

- Morphisms: Hom, Iso, Equivalence

- Theorems: UniversalProperties, Classification, Main
- Examples: Standard, Counterexamples
- Bridges: ToAlgebra, ToTopology, ToGeometry
